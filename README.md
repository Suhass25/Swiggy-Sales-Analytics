# 🍔 Swiggy Sales Data Analysis

## 📌 Project Overview
This project explores a real-world Swiggy dataset to uncover insights regarding restaurant performance, cuisine popularity, and revenue generation. The analysis was conducted fully in Microsoft Excel to demonstrate strong foundational data manipulation and reporting skills.

## 🛠️ Tools & Techniques Used
* **Microsoft Excel:** * **Data Wrangling:** Used Text-to-Columns and Power Query tools to format messy restaurant data.
  * **Trend Analysis:** Created Pivot Tables to track total sales volume across different days of the week.
  * **Metrics Calculation:** Used aggregation formulas (SUMIFS, COUNTIFS) to calculate average order values and total revenue per restaurant.
  * **Data Visualization:** Designed Bar Charts and Pie Charts to present findings clearly.
